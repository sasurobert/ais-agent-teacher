# AIS Agent Teacher

## Project Overview
The **AIS Agent Teacher** is an efficiency-driven personal assistant for educators within the AISchoolOS ecosystem. Unlike the student-facing tutor, this agent focuses on direct output, rapid drafting, and proactive classroom insights.

## Key Features
- **Efficiency Persona**: Provides direct answers, lesson hooks, and feedback drafts.
- **Proactive Monitoring**: Flags students who over-abuse help buttons or struggle with foundational concepts.
- **Curriculum Alignment**: Generates Bible-aligned analogies and Socratic questions for teachers to use in class.
- **Integration**: Seamlessly connects with `Edu Service` and Google Classroom.

## Tech Stack
- Node.js / TypeScript
- LangGraph (Agentic Reasoning)
- Prisma (Data Sync)
- LangChain

## Getting Started
1. `npm install`
2. `npx prisma generate`
3. `npm run dev`
